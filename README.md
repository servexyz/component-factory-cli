# Component Generator CLI

> Purpose of this is to create scaffolding for multiple components easily

Input:
cgc componentName componentNamely componentNameth

Output:
```
./componentName
-- componentName.js
-- componentName.test.js
-- index.js
./componentNamely
-- componentNamely.js
-- componentNamely.test.js
-- index.js
./componentNameth
-- componentNameth.js
-- componentNameth.js
-- index.js
```
